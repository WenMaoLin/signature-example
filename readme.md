# Signature 签字组件 example

工程基于 octopus + nusi-mobile，展示如何在 Form 中使用签字组件。

## Run

```shell
yarn
yarn watch:wechat
```

在微信开发者工具中将 dist/wx 目录作为项目路径导入即可预览效果
